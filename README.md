# Task Manager App

A clean and minimal task manager app built with vanilla HTML, CSS, and JavaScript.

## Features

- Add tasks via a modal popup
- Mark tasks as complete (with strikethrough)
- Delete tasks
- Timestamps on every task
- Tasks persist across page refreshes with localStorage
- Responsive design for mobile and desktop

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

## Getting Started

Clone the repo and open `index.html` in your browser — no setup required.

```bash
git clone https://github.com/your-username/task-manager-app.git
cd task-manager-app
```

Then just open `index.html` in any browser.
